### VARIABLES DE CONFIGURATION

* `BOT_TOKEN`  - Obtenez le token de votre bot via <a href="https://t.me/BotFather" target="/blank">Bot Father</a>
* `API_ID` - Obtenez votre api id sur <a href="https://my.telegram.org" target="/blank">Telegram Auth</a>
* `API_HASH` - Obtenez votre api hash sur <a href="https://my.telegram.org" target="/blank">Telegram Auth</a>
* `ADMIN` - ID utilisateur de l'admin
* `LOG_CHANNEL` - Canal de logs du Bot ⚠️ L'id doit commencer par -100.
* `SUPABASE_URL`  - URL de votre projet Supabase (Project Settings > API)
* `SUPABASE_KEY`  - Clé API de votre projet Supabase
* `FORCE_SUBS` - Le nom d'utilisateur de votre canal d'abonnement forcé, sans @ `Optionnel`
* `START_PIC` - Photo du message de démarrage. `Optionnel`

### TOUTES LES COMMANDES

```
start - Vérifier si le bot fonctionne.
viewthumb - Voir la miniature actuelle.
delthumb - Supprimer la miniature actuelle.
set_caption - Définir une légende personnalisée.
see_caption - Voir votre légende personnalisée.
del_caption - Supprimer la légende personnalisée.
metadata - Modifier vos métadonnées.
ping - Vérifier le ping du bot.
donate - Soutenir le développeur.
set_prefix - Définir votre préfixe
see_prefix - Voir votre préfixe
del_prefix - Supprimer votre préfixe
set_suffix - Définir votre suffixe
see_suffix - Voir votre suffixe
del_suffix - Supprimer votre suffixe
restart - Redémarrer le bot [RÉSERVÉ AUX ADMINS]
broadcast - Commande de diffusion de message [RÉSERVÉ AUX ADMINS].
status - Vérifier le statut du bot [RÉSERVÉ AUX ADMINS].
```
